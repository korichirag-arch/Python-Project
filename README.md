# 🚀 Fundamental Booster Program

A simple yet interactive Python command-line application designed to collect, analyze, and display user data along with its core Python attributes (Data Types and Memory Addresses). This project serves as a practical demonstration of fundamental Python concepts such as dynamic typing, user input handling, and object memory IDs.

## ✨ Features
- **Interactive CLI**: A welcoming and user-friendly command-line interface.
- **Data Type Inspection**: Automatically detects and displays the data type (`type()`) of each user input.
- **Memory Address Mapping**: Reveals the exact memory address (`id()`) where Python stores the variables.
- **Dynamic Calculation**: Dynamically calculates the user's approximate birth year based on the current year (2026).

## 🛠️ Concepts Covered
- Standard Input/Output (`input()` and `print()`)
- Type Casting (`int()`, `float()`)
- String Formatting (f-strings)
- Core Python Functions (`type()` and `id()`)
